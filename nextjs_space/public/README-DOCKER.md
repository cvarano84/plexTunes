# Plex Jukebox - Docker Setup

A premium TouchTunes-style jukebox interface for your Plex music library.

## Quick Start

### 1. Clone and configure

```bash
cp .env.example .env
# Edit .env with your API keys
```

### 2. Build and run

```bash
docker-compose up -d
```

### 3. Access the app

Open `http://localhost:3000` in your browser.

### 4. Initial Setup

1. Enter your Plex server URL (e.g., `http://192.168.1.100:32400`)
2. Enter your Plex authentication token
3. Select your music library
4. Wait for the initial sync to complete

## Environment Variables

| Variable | Description | Required |
|----------|-------------|----------|
| `DATABASE_URL` | PostgreSQL connection string | Yes (auto-configured in Docker) |
| `GENIUS_ACCESS_TOKEN` | Genius API token for lyrics | Yes |
| `SPOTIFY_CLIENT_ID` | Spotify app client ID | Yes |
| `SPOTIFY_CLIENT_SECRET` | Spotify app client secret | Yes |

## Finding Your Plex Token

1. Sign in to Plex Web App
2. Navigate to any library item
3. Click the `...` menu → Get Info
4. In the URL bar, look for `X-Plex-Token=YOUR_TOKEN_HERE`

## Features

- 🎵 **Smart Stations** - Auto-generated by decade and genre from your library
- 🎤 **Lyrics Display** - Synchronized lyrics powered by Genius
- ⭐ **Popularity Filtering** - Shows popular hits first using Spotify data
- 📱 **Touch Optimized** - Large buttons, touch-friendly gestures
- 🎨 **Beautiful UI** - Dark theme with neon accents, album art showcase
- 🔍 **Smart Search** - Search across artists, albums, and tracks
- 📀 **Queue Management** - Build and manage your playback queue

## Persistent Data

All data is stored in the PostgreSQL database:
- Plex server credentials
- Cached library metadata
- Station configurations
- Spotify popularity scores

The `pgdata` Docker volume ensures data survives container restarts.

## Updating

```bash
docker-compose pull
docker-compose up -d --build
```

## Troubleshooting

- **Can't connect to Plex**: Ensure your Plex server is reachable from the Docker container
- **No stations appearing**: Make sure to sync your library first, stations require 5+ matching tracks
- **No lyrics**: Genius API may not have lyrics for all songs
