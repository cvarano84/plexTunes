# Plex Jukebox - Project Instructions

## Overview
A premium TouchTunes-style jukebox web app for Plex music libraries. Dark-themed, touch-optimized interface for large screens.

## Tech Stack
- Next.js 14, React 18, TypeScript, Tailwind CSS
- PostgreSQL via Prisma ORM
- External APIs: Plex (music library/streaming), Genius (lyrics), Spotify (popularity data)

## Design Language
- **Theme**: Dark, always-dark (no light mode). CSS variables set for dark jukebox aesthetic.
- **Colors**: Primary = purple (270 80% 60%), Accent = cyan (195 90% 50%), Background = deep navy (230 25% 7%)
- **Effects**: Neon text glow, album art glow shadows, vinyl spin animation, lyrics scroll mask
- **Touch**: Large click targets, touch-friendly, optimized for large screens
- **Fonts**: DM Sans (body), Plus Jakarta Sans (display), JetBrains Mono (mono)

## Architecture
- `/setup` - First-time Plex credential entry + library sync
- `/jukebox` - Main SPA-style interface with nav tabs: Stations, Artists, Search, Now Playing, Queue
- Client-side `PlayerProvider` context manages audio playback state
- All Plex communication proxied through API routes to keep token server-side

## Database Models
- `PlexConfig` - Stored Plex server URL and token
- `CachedArtist/Album/Track` - Full library cache from Plex
- `Station` - Auto-generated stations by decade+genre
- `LibrarySyncStatus` - Sync progress tracking

## API Routes
- `/api/plex/config` - GET/POST Plex credentials
- `/api/plex/library` - GET music library sections
- `/api/plex/sync` - POST to sync library, GET for progress
- `/api/plex/stream` - Proxy audio streaming
- `/api/plex/image` - Proxy album art images
- `/api/artists` - Paginated artist list with search
- `/api/artists/[id]` - Artist detail with albums/tracks
- `/api/albums/[id]` - Album detail with tracks
- `/api/stations` - List active stations
- `/api/stations/[id]/tracks` - Get shuffled station tracks
- `/api/search` - Global search across artists/albums/tracks
- `/api/lyrics` - Genius lyrics lookup
- `/api/popularity` - Batch Spotify popularity updates

## Key Decisions
- Always dark theme (class="dark" on html)
- Station generation requires 5+ matching tracks to be visible
- Spotify popularity cached in DB to avoid repeated API calls
- Images proxied through server to keep Plex token secure
- Docker files stored in public/ for easy download

## External API Secrets
- GENIUS_ACCESS_TOKEN - Genius lyrics API
- SPOTIFY_CLIENT_ID / SPOTIFY_CLIENT_SECRET - Spotify client credentials flow
